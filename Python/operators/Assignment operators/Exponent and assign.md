- Use the `**` operator to obtain the exponent (raise power) value, and then assign result.
syntax:
```Python
x = x ** y
```

Example:
```Python
a, b = 2, 3
print(a, b)
a = a ** b
print(a)
```

- Calculate the exponent (raise power) value using operands and then assigning the result to the left operand.
```Python
x **= y
```

Example:
```Python
a, b = 2, 3
print(a, b)
# a = a ** b
a **= b
print(a)
```

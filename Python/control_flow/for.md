Iterate over the items of a sequence, such as a [[lists|list]], [[tuples|tuple]] or a [[Python/data types/strings/strings|string]].

syntax:
```Python
for <iterating_var> in <sequence>:
	<suite>
```

Example using string:
```Python
myString = "Foo"
for i in myString:
	print(i)
```

Example using list:
```Python
myList = ['Foo', 'Bar']
for i in myList:
	print(i)
```

- Use the `-` operator to substract values, and then assign result.
syntax:
```Python
x = x - y
```

Example:
```Python
a, b = 2, 3
print(a, b)
a = a - b
print(a)
```

- Subtract the right operand from the left operand and then assign the result to the left operand.
syntax
```Python
x -= y
```

Example:
```Python
a, b = 2, 3
print(a, b)
# a = a - b
a -= b
print(a)
```

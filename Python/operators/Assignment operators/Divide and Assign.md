- Use the `/` operator to divide values, and then assign result.
syntax:
```Python
x = x / y
```

Example:
```Python
a, b = 2, 3
print(a, b)
a = a / b
print(a)
```

- Divide the left operand with the right operand and then assig the result to the left operand.
syntax
```Python
x /= y
```

Example:
```Python
a, b = 2, 3
print(a, b)
# a = a / b
a /= b
print(a)
```

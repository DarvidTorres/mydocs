- Use the `+` operator to add values, and then assign result.
syntax:
```Python
x = x + y
```

Example:
```Python
a, b = 2, 3
print(a, b)
a = a + b
print(a)
```

- Add right side operand with left side operand and then assign to left operand
syntax
```Python
x += y
```

Example:
```Python
a, b = 2, 3
print(a, b)
# a = a + b
a += b
print(a)
```
